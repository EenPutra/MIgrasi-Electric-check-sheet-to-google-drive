Weekly Report Dashboard - Maintenance EIC7

Tool input data mingguan: Excel WO → update & foto → export untuk dibuatkan PPT oleh Claude

Cara Pakai

1. Pilih sheet di bawah: Weekly Report untuk laporan mingguan biasa (import Excel WO dari CMMS), atau Weekly Job Arrangement untuk data planning/penugasan mingguan (mis. dari PDF Daily Plan Activity). Kedua sheet punya fitur & input yang sama tapi datanya disimpan terpisah.
2. Klik Import Excel WO, pilih file export List of Work Order dari CMMS (format sama seperti sebelumnya), atau lanjutkan dari file .json yang pernah di-export.
3. Data otomatis dikelompokkan: Completed / Outstanding / Cancelled, plus ringkasan KPI di bawah.
4. Klik satu baris WO untuk buka detail → isi/edit kolom Update (What was Complete?), centang Highlight untuk laporan jika WO tsb perlu ditampilkan di PPT, dan upload foto evidence bila ada.
5. Isi catatan Rekomendasi & Rencana Tindak Lanjut di bagian bawah bila ingin menambahkan poin manual.
6. Klik Export Data (.json) → upload file hasil export ini ke chat Claude, lalu minta: "buatkan laporan PPT minggu ini dari data ini". Foto ikut tersimpan di dalam file JSON, jadi cukup 1 file yang diupload.
Data tersimpan otomatis di browser ini (draft) selama tidak clear cache, per sheet masing-masing. Selalu Export ke .json setiap selesai kerja agar tidak hilang & bisa dipakai lintas perangkat.
