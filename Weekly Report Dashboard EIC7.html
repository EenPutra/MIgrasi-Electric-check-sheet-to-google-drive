Weekly Report Dashboard - Maintenance EIC7

Tool input data mingguan: Excel WO → update & foto → export untuk dibuatkan PPT oleh Claude

Cara Pakai

1. Klik Import Excel WO, pilih file export List of Work Order dari CMMS (format sama seperti sebelumnya).
2. Data otomatis dikelompokkan: Completed / Outstanding / Cancelled, plus ringkasan KPI di bawah.
3. Klik satu baris WO untuk buka detail → isi/edit kolom Update (What was Complete?), centang Highlight untuk laporan jika WO tsb perlu ditampilkan di PPT, dan upload foto evidence bila ada.
4. Isi catatan Rekomendasi & Rencana Tindak Lanjut di bagian bawah bila ingin menambahkan poin manual.
5. Klik Export Data (.json) → upload file hasil export ini ke chat Claude, lalu minta: "buatkan laporan PPT minggu ini dari data ini". Foto ikut tersimpan di dalam file JSON, jadi cukup 1 file yang diupload.
Data tersimpan otomatis di browser ini (draft) selama tidak clear cache. Selalu Export ke .json setiap selesai kerja agar tidak hilang & bisa dipakai lintas perangkat.
